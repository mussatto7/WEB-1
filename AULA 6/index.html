Flex Box

são caixas flexiveis que podem se adaptar aos Breakpoints declarados no CSS - É como se fosse uma tabela.

Elementos e atributos trabalhados junto com o Flex Box:

      Display: Exibe um elemento
      Flex-direction: direção do elementos, por padrão declaramos ele em row (linhas)
      Flex-wrap: define quebra de linha da caixa
      Flex-flow: atalho para o uso do direction e wrap
      Justify-content: conteudo justificado
      align-intems: alinhamento dos itens dentro do elemento
